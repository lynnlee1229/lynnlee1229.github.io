【经验分享】合并多个jupyter notebook文件

Posted by Lynn on Thursday, October 21, 2021

前言

博主在用jupyter notebook做学习笔记的时候，发现了他人train好的一个网络的.ipynb文件，由于源代码训练时间过长，故想直接将两文件合并，遂研究合并方法。

nbmerge

人生苦短，you need python…搜索了一下，果然有工具啊。下面简单介绍一下~

安装

pip install nbmerge

使用

以博主的两文件需求为例，假设待合并文件名为1.ipynb、2.ipynb，合并后文件名为3.ipynb。 首先，打开待合并文件所在目录，shift+鼠标右键在此处打开powershell。 然后输入如下命令

nbmerge 1.ipynb 2.ipynb > 3.ipynb

完成合并，如图，将2中的训练结果直接合并到我1文件中的学习笔记中，美滋滋~ 以上部分只要搜nbmerge都可以看到类似的使用方法介绍，接下来才是我发这个博客的原因。

出现的问题

可以看到我的3文件合并后非常大，远大于1+2的文件大小。并且此时尝试打开合并后的文件也会出现错误。我在网上简单检索未发现类似问题的回答。 遂研究了一下其报错信息，如下图所示。 我认为这是编码解析问题，用notepad打开查看，果然。。 更改其为UTF-8编码并保存，可以看到，3文件的大小基本等于1+2了，并且此时3文件也可以正常打开了。 没什么技术含量，就是对一个无人解答问题的经验分享，希望能帮助到遇到类似问题的人~
